# Playcanvas Asset Processor

Это настольное приложение на WPF предназначено для работы с проектами PlayCanvas. Оно позволяет подключаться к вашему аккаунту, загружать файлы проекта и управлять ресурсами.

## Возможности

- Загрузка информации о проектах и ветках через API PlayCanvas
- Скачивание текстур, моделей и материалов
- Проверка целостности файлов
- Просмотр изображений и моделей внутри приложения

## Требования

- **.NET 9.0** (предварительная версия)
- Visual Studio 2022 Preview или аналогичная IDE

## Сборка

```bash
# клонируем репозиторий
git clone https://github.com/username/PlaycanvasAssetProcessor.git
cd PlaycanvasAssetProcessor

# открываем решение `TexTool.sln` в Visual Studio и собираем проект
```

## Запуск

После сборки откройте `TexTool.exe`. Перед первым запуском укажите ваш API‑ключ PlayCanvas и другие параметры в окне настроек.
